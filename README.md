# e-plantShopping

## Paradise Nursery Shopping Application

e-plantShopping is a React-based shopping cart application for an online plant store called Paradise Nursery. The application allows users to browse a variety of house plants, add them to a shopping cart, adjust item quantities, and view the total cost before checkout.

### Features
- Landing page with background image and Get Started button
- Product listing page with categorized house plants
- Add to Cart functionality with dynamic cart count
- Shopping cart page with quantity controls and delete option
- Redux Toolkit used for state management
- Responsive navigation bar across pages

### Technologies Used
- React
- Redux Toolkit
- React Router
- JavaScript
- HTML & CSS

### Project Purpose
This project is part of the final assessment for the React course and demonstrates component-based architecture, state management with Redux, and dynamic UI updates.
